手续费

矿工费用取决于底层区块链以及当时网络的吞吐量。通过 Solana 的核心技术 ，在 Serum DEX 中平均每笔交易的网络手续费大约为0.00001 美元。这意味着目前在 Bonfida 上空投的 0.004SOL（大约价值 0.00764 美元），理论上可以满足 764 笔交易的手续费。

当您进行以下操作时，将产生矿工费用：

  1. 发出订单
  2. 取消订单
  3. 交易后结算资金
  4. 向基于 Solana 的钱包发送/充值资金
